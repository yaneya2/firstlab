# Первая лабораторная работа по ПАТД

## Общее описание

Проект представляет собой реализацию **универсального динамического массива (DynamicArray)** на языке C с поддержкой полиморфизма через систему аллокаторов. Массив может работать с различными типами данных: `double`, `string` и пользовательской структурой `Point` (трёхмерные точки).

Программа предоставляет интерактивный интерфейс командной строки для управления динамическими массивами и выполнения операций над ними.

### Компоненты

| Файл | Описание |
|------|----------|
| `DynamicArray.h/c` | Структура `DynamicArray` и все операции: создание, добавление, получение, сортировка, map, where, concat |
| `FieldInfo.h/c` | Полиморфная система описания типов: размер, сравнение, аллокация, копирование |
| `UI.h/c` | Три режима интерфейса: `double_ui()`, `string_ui()`, `point_ui()` |
| `UsersStruct.h` | Пользовательская структура `Point { x, y, z }` |
| `Allocator.h` | Типы функций для выделения и освобождения памяти |

## Функциональность

Динамический массив поддерживает следующие операции:

| Операция | Описание |
|----------|----------|
| **create** | Создание пустого динамического массива для указанного типа |
| **add** | Добавление элемента в конец массива |
| **get** | Получение элемента по индексу |
| **set** | Изменение значения элемента по индексу |
| **sort** | Сортировка элементов массива |
| **concat** | Объединение двух массивов одного типа |
| **where** | Фильтрация элементов по предикату |
| **map** | Преобразование элементов функцией |
| **print** | Вывод всех элементов массива |
| **destroy** | Освобождение всей выделенной памяти |

---

## Поддерживаемые типы данных

| Тип | Описание |
|-----|----------|
| `double` | Числа с плавающей запятой двойной точности |
| `string` | Строки переменной длины (`char*`) |
| `Point` | Трёхмерные точки с координатами `(x, y, z)` |

Каждый тип описывается структурой `FieldInfo`, содержащей:
- Размер элемента в байтах
- Функцию сравнения (для сортировки)
- Функцию выделения памяти (для строк)
- Функцию освобождения памяти
- Функцию присваивания значений

## Интерфейс пользователя

Проект предоставляет три режима работы:

```c
void double_ui();   // Работа с числами double
void string_ui();   // Работа со строками
void point_ui();    // Работа с трёхмерными точками
```

По умолчанию в `main.c` вызывается `double_ui()`. Для смены режима измените вызов в `main.c`:

```c
#include "headers/UI.h"

int main(void) {
    point_ui();  // или string_ui()
}
```

## Особенности реализации

- **Полиморфизм на C** — система `FieldInfo` позволяет работать с разными типами через единый интерфейс
- **Автоматическое расширение** — вместимость массива увеличивается при необходимости
- **Безопасность памяти** — вся выделенная память корректно освобождается
- **Обработка ошибок** — проверка граничных условий и возврат статусов операций
- **Расширяемость** — легко добавить новый тип данных, реализовав `FieldInfo`
